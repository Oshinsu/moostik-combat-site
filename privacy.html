Politique de confidentialité

Dernière mise à jour :

La présente politique décrit comment BYSS GROUP (l'« Éditeur ») traite les données dans le cadre du jeu Moostik Combat (le « Jeu »), disponible sur le web et via les stores d'applications.

2. Le principe : un jeu sans compte et minimaliste

Moostik Combat ne demande aucune inscription et ne collecte aucune donnée d'identification (nom, e-mail, téléphone). Tu peux jouer entièrement de façon anonyme.

3. Données traitées

  • Pseudo (facultatif) : si tu en saisis un pour le classement, il est enregistré tel quel, associé à tes records de jeu (victoires, meilleure série, KO le plus rapide). Aucune autre donnée n'y est liée.
  • Données de partie : scores et statistiques de jeu, stockées localement sur ton appareil (stockage local du navigateur) et, si tu utilises le classement, sur notre prestataire d'hébergement de base de données.
  • Mode en ligne (multijoueur) : pour établir une connexion directe entre deux joueurs (WebRTC), des informations techniques de connexion (dont l'adresse IP) sont échangées via un serveur de mise en relation. Ces informations sont transitoires et ne sont pas conservées par l'Éditeur.

Le Jeu n'affiche pas de publicité et n'utilise pas de traceur publicitaire ni de cookie de suivi marketing.

4. Sous-traitants / hébergeurs

  • Supabase — hébergement de la base de données du classement (pseudo + records).
  • Railway — serveur de mise en relation pour le multijoueur (relai de connexion, sans logique de jeu).
  • Hébergement du site / de l'application web (CDN).

Ces prestataires peuvent traiter des données techniques pour le seul fonctionnement du service.

5. Finalités & base légale

Les données ne servent qu'au fonctionnement du jeu (afficher un classement, connecter deux joueurs). Base légale : l'intérêt légitime de fournir le service et, pour le pseudo du classement, ton action volontaire (consentement).

6. Durée de conservation

Les records de classement sont conservés tant que le classement existe. Le profil local reste sur ton appareil jusqu'à effacement des données du navigateur/de l'app. Les informations de connexion en ligne ne sont pas conservées.

7. Tes droits (RGPD)

Tu disposes d'un droit d'accès, de rectification et d'effacement. Pour exercer ces droits ou supprimer ton pseudo du classement, écris à contact@byss-group.com.

8. Public

Moostik Combat est un jeu de combat stylisé, tout public (classification indicative PEGI 7-12). Il ne contient pas de collecte de données ciblant les enfants.

9. Modifications

Cette politique peut évoluer ; la date de mise à jour ci-dessus fait foi.
